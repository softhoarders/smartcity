# Hardware pricing (Romania, B2B operators)

Recommended **one-time kit price per bay** for Spotflow edge monitoring (Raspberry Pi 5 + camera + enclosure + SD + power), sold to parking operators or building managers — not end drivers.

## Bill of materials (approximate, retail RON, Q1 2026)

| Component | Typical cost (RON) |
|-----------|-------------------|
| Raspberry Pi 5 (4 GB) | 350–420 |
| Camera module (IMX477 / HQ or quality USB) | 180–350 |
| Weather-resistant enclosure + mount | 120–220 |
| microSD 64 GB (industrial preferred) | 50–90 |
| PSU + USB-C cable + surge protection | 80–120 |
| Misc. (cabling, labels, consumables) | 40–80 |

**Hardware subtotal:** roughly **820–1,280 RON** per bay before logistics.

## Install labor

| Item | Typical cost (RON) |
|------|-------------------|
| Site survey + mounting (1 bay, urban) | 150–300 |
| Network join + Spotflow pairing + test | 100–200 |

**Labor subtotal:** roughly **250–500 RON** per bay for a straightforward install.

## Recommended operator price (B2B)

| Tier | One-time per bay (RON) | Notes |
|------|------------------------|--------|
| **List / MSRP** | **1,650–1,950** | Includes hardware kit, standard enclosure, 12-month hardware warranty, remote onboarding |
| **Volume (10+ bays)** | **1,450–1,750** | Shared site visit, bulk shipping |
| **Volume (50+ bays)** | **1,250–1,550** | Operator self-install option with remote support |

These figures leave modest margin after retail BOM and one install visit, without undercutting professional integrators. Adjust upward if using PoE, LTE backup, or premium housings (+200–400 RON/bay).

## What is not included

- Monthly SaaS / platform fee (priced separately).
- Civil works, trenching, or permit costs.
- Ongoing cellular data plans.
